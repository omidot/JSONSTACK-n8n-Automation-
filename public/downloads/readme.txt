JSONSTACK WORKFLOWS DOWNLOAD FOLDER
===================================

This folder (/public/downloads/) contains the files that customers will be able to download AFTER they pay successfully.

Instructions for you (Vendor):
1. Create a ZIP file of your 300+ workflows.
2. Rename your ZIP file to: `jsonstack-bundle.zip`
3. Drag and drop or upload that `jsonstack-bundle.zip` file directly into this `/public/downloads/` folder using the File Explorer in your workspace editor.
4. Once uploaded, any customer who pays successfully via PayPal (status = Completed) will be able to click "Download' and obtain this real zip file straight from your self-hosted setup!

Note:
The application's code is configured to grab the file from the path `/downloads/jsonstack-bundle.zip` automatically when the customer clicks any download button after successful payment.
